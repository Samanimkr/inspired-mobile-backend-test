## Steps to start server

#### Using yarn:
1. Run `yarn install`
2. Run `yarn start`

#### and using npm:
1. Run `npm install`
2. Run `npm start`

## Endpoints

`GET  /users/` - Get all users

`POST /users/` - Create a user

`GET  /users/:userId` - Get user by id

`PUT  /users/:userId` - Update user by id

`DELETE /users/:userId` - Delete user by id
